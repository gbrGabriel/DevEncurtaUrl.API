# DevEncurtaUrl.API

Foi desenvolvida uma API REST completa para encurtamento de links

## Tecnologias e práticas utilizadas
- ASP.NET Core com .NET 7
- Entity Framework Core
- SQL Server
- Swagger
- Injeção de Dependência
- Programação Orientada a Objetos
- Serialog
- REST

## Funcionalidades
- Cadastro de link  
- Listagem de links
- Editar dados do link cadastrado
- Apagar link

## Passo a passo para rodar o Projeto (Visual Studio)
- Necessário SDK .NET 7.0 
- (Caso tenha um banco local e queira usar, basta alterar a connectionString no appsettings.json)
##
- Faça o build da aplicação para realizar o restore dos pacotes,
- logo depois rode o comando no PMC Update-Database
